# URL Health Checker

This is a Streamlit-based Python application that extracts URLs from pasted or uploaded text and checks their HTTP/HTTPS status codes. It supports live interaction and displays results in a table. Ideal for verifying the health of multiple links in a document or text snippet with ease and clarity.

## Features
- Extracts all URLs from text or uploaded file
- Checks HTTP/HTTPS status codes
- Interactive Streamlit interface
- Displays results in a sortable table

## Usage
Run the application using:
```bash
streamlit run main.py
```
